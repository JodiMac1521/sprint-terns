## 🧩 What is Sprint-Terns™?

**Sprint-Terns™** is an AI-powered workforce platform by [Work Simplr](https://worksimplr.com) that connects businesses with structured sprint teams for rapid project execution.

This repo is the **customer-facing marketing site** — built from scratch in React with a custom dark design system, full routing, and CI/CD via GitHub Actions.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + React Router v6 |
| Build tool | Vite 6 |
| Styling | Custom CSS — CSS variables, animations, no UI library |
| Typography | Syne (display) + DM Sans (body) |
| Deployment | GitHub Actions → GitHub Pages |
| Node | v22 |

## 🚀 Getting Started

```bash
git clone https://github.com/JodiMac1521/sprint-terns.git
cd sprint-terns
npm install
npm run dev
# → http://localhost:5173
```
